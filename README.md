# Week 4 Internship Projects

## Projects Included
1. Basic Attack Simulation
2. Post-Hardening Nmap Scan
3. Final Report

## 1. Basic Attack Simulation

### Objective
Simulate a basic attack on a test environment and document response.

### Tasks Performed
- Selected target system  
- Performed simulated attack  
- Recorded system behavior and response steps  

### Tools Used
- Lab Environment  

### Result
Understood attack-response workflow and incident documentation.

## 2. Post-Hardening Nmap Scan

### Objective
Re-scan the system after hardening to verify security improvements.

### Tasks Performed
- Performed Nmap scan on hardened system  
- Compared results with pre-hardening scan  
- Documented changes in open ports and services  

### Tools Used
- Nmap  

### Result
Confirmed security hardening reduced open ports and potential vulnerabilities.

## 3. Final Report

### Objective
Compile all internship work into a final report with findings and risk classification.

### Tasks Performed
- Documented weekly tasks  
- Added screenshots and explanations  
- Classified findings by risk severity  

### Tools Used
- Word / PDF  

### Result
Complete internship documentation ready for submission.
